Assignments 1–4 embedded · all original interactions preserved
Overview

What this page contains: This page works as a single hub for my IDV work, embedding Assignments 1–4 into one combined view. You can switch between them using the navigation bar below.

How the assignments are merged: Each assignment is kept as an independent page and loaded inside this layout through an iframe. This keeps all original code, transitions, and interactions unchanged, while giving a unified outer frame for navigation.

Main difficulties: The tricky parts were aligning different page layouts inside a single frame, hiding scrollbars at the outer level while still allowing smooth scrolling, and making sure the embedded pages remained responsive without breaking their existing animations.

Hardest assignment to integrate: Assignment 1 was the hardest to embed because it relies on live data and more complex scripting logic. I had to make sure that loading it inside an iframe would not interfere with its API calls, timing, and transition effects.

Tip: Use your mouse wheel / trackpad to scroll the page vertically. Scrollbars are hidden visually, but full-page scrolling and all assignment interactions remain available.
